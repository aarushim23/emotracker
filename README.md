# EmoTrack - Emotion Tracker

A Flask-based emotion tracking application powered by DistilBERT. Analyzes text input to detect emotions (joy, sadness, anger, fear, love, surprise) and provides insights, trends, and recommendations.

## Features
- Real-time emotion prediction from text
- Emotion timeline tracking
- Trend analysis (24h and 7-day)
- AI-generated insights and recommendations
- CSV tweet upload for bulk analysis
